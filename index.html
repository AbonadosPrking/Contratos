Cómo subir la app a Dropbox

1
Subir el archivo a Dropbox
Ve a dropbox.com, haz clic en "Subir archivos" y selecciona contrato_parktech_v3.html. Guárdalo en una carpeta accesible, por ejemplo Herramientas/.
2
Obtener enlace público
Haz clic en los tres puntos (⋯) junto al archivo → "Compartir" → "Copiar enlace". El enlace tendrá este formato:
https://www.dropbox.com/s/XXXX/contrato.html?dl=0
3
Convertir a enlace directo
Cambia el final ?dl=0 por ?dl=1&raw=1 para que el navegador lo abra directamente en lugar de descargarlo:
https://www.dropbox.com/s/XXXX/contrato.html?dl=1&raw=1
4
Pegar la URL en la app
Abre el HTML en el Bloc de notas, busca la línea:
// URL pública del archivo en Dropbox (se configura en el modal de ajustes) function getBaseURL() { // Check sessionStorage for configured Dropbox URL const saved = sessionStorage.getItem('parktech_base_url'); if (saved && saved.startsWith('http')) return saved; // Fallback: current URL (only works if already hosted) const cur = window.location.href.split('?')[0]; if (cur.startsWith('http') && !cur.startsWith('file')) return cur; return null; }y reemplázala por:
const BASE_URL = 'https://www.dropbox.com/s/XXXX/contrato.html?dl=1&raw=1';Vuelve a subir el archivo actualizado a Dropbox.
5
¡Listo! Cada vez que el departamento rellene la Fase 1 y pulse "Generar enlace", se creará un enlace único para ese cliente. El cliente lo abrirá desde su móvil u ordenador y aterrizará directamente en la Fase 2.
